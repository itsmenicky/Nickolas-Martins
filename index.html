NerdVision
Site para Ótica em ReactJS

Projeto no qual desenvolvi meu primeiro carrinho de compras em ReactJS. Criei o website de uma ótica fictícia com os itens sendo carregados dinamicamente, se utilizando do hook useState para armazenar quais itens estão ativos no carrinho e useEffect para estar atualizando o valor total da compra. Com este projeto aperfeiçoei minhas habilidades com React, mexendo com conceitos básicos e avançados dessa poderosa ferramenta.
